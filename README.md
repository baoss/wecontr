# wxcms

微信小程序wxcms前台代码


## 资讯文章类小程序

玩法：阅读、点赞文章获得积分，积分用于在积分商城兑换现金会员等礼品。
获得礼品需要添加微信客服为好友，发送兑换码完成。

通过完成任务获得积分，积分在商城兑换成卷，添加客服微信好友后凭卷密令提现。
